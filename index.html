Descripción del problema

En muchas viviendas, edificios y pequeñas industrias, el control del nivel de agua en los tanques se realiza de forma manual. Esto provoca desperdicio de agua, daños en las bombas por funcionamiento en seco y falta de información en tiempo real sobre el estado del tanque.

La ausencia de un sistema automatizado genera:

  • Riesgo de quedarse sin agua en momentos críticos.
  • Desperdicio de recursos por reboses constantes.
  • Mayor consumo eléctrico por encendido y apagado manual de bombas.

Explicación de la solución

La solución propuesta es un sistema IoT de monitoreo de nivel de agua que integra sensores de nivel, una placa microcontroladora y una plataforma en la nube para registrar y visualizar los datos en tiempo real.

El sistema envía notificaciones cuando el tanque está por vaciarse o por rebosar, permitiendo tomar decisiones rápidas y automatizar el encendido de la bomba de agua.

Objetivos principales:
  • Monitorear el nivel de agua de forma continua.
  • Reducir el desperdicio de agua y energía.
  • Informar al usuario mediante alertas (por ejemplo, correo o app).

El funcionamiento del sistema

El sistema se compone de un tanque de agua con sensores de nivel, una placa controladora (por ejemplo, Arduino o ESP32) y un módulo de conectividad a Internet.

  1. Los sensores detectan el nivel actual de agua (bajo, medio, alto).
  2. La placa lee los datos y los envía a la nube mediante Wi-Fi.
  3. En la plataforma en la nube se almacenan los datos y se muestran en gráficos.
  4. Cuando el nivel es muy bajo o muy alto, se genera una alerta para el usuario.
  5. Opcionalmente, el sistema puede encender o apagar la bomba de forma automática.

Tecnologías utilizadas e impacto

Tecnologías utilizadas

  • Placa microcontroladora: Arduino / ESP32.
  • Sensores de nivel: flotadores o sensores ultrasónicos.
  • Conectividad: Wi-Fi para enviar datos a la nube.
  • Plataforma en la nube: base de datos y panel web de monitoreo.
  • Interfaz web: página informativa para el usuario (HTML y CSS).

Impacto

  • Uso más eficiente del agua.
  • Disminución de daños en la bomba.
  • Mayor comodidad para el usuario.
  • Posibilidad de escalar a edificios, conjuntos residenciales o industrias.
